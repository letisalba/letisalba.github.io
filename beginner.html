Beginner:

This is one of two homework assignments you can complete for this week. Your assignment should be in the form of an HTML page, which I will be able to run locally. Refer to the file 'js_in_webpage.html' on the github page for instructions on setting up javascript to run on an HTML page, and how to run / debug.

Question 1: Create a function to reverse any word that you type in. This can be typed into either an input box or an alert box, and then print in a box or on the webpage.


Question 2: Create a function that takes an input number, and prints a table with the first 20 multiples of the number in order 5x4.


Intermediate:

Write a function to load the presidents.csv data and display the data as an html table (you may use a cdv parser from any available library, though I encourage you to try the d3 one).

Now that you have the presidents data loaded, write a function that will return the president's height & weight when you type it in. This can be typed into either an input box or an alert box, and then print the result in a box or on the webpage.

Deployment: Your apps from part 1 and 2 must be deployed. There are a lot of options for deploying javascript apps, however I would recommend using github pages. Read more about how github pages work here:https://pages.github.com/
